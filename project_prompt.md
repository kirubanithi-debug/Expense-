### 🏗️ 2. Architectural Blueprint (High-Tech Stack)
*   **Frontend Core:** Vite + React 18, React Router Dom (SPA architecture).
*   **State Management:** Decentralized Context API system (Auth, Expenses, Notification) for scalability and performance.
*   **Backend Ecosystem:** Node.js Express server with a high-performance JSON-based persistence layer for local database agility.
*   **Data Visualization:** Chart.js + React-Chartjs-2 for high-density, real-time financial trajectory modeling.
*   **Voice Core:** Web Speech API integration for the "Power Mic" UI, featuring NLP (Natural Language Processing) parsing of expenditure requests.
